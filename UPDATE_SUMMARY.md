## 🎯 What You Need to Do

### To Run the Application:

1. **Install Node.js** (if not already installed)
   - Download: https://nodejs.org/

2. **Open terminal in project folder**
   ```bash
   cd marketing-studio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Add your API key**
   - Open `.env` file
   - Replace: `GEMINI_API_KEY=your_actual_api_key_here`
   - With: `GEMINI_API_KEY=AIzaSyC_your_real_key`
   - Get key: https://makersuite.google.com/app/apikey

5. **Start server**
   ```bash
   npm start
   ```

6. **Open browser**
   - Go to: http://localhost:3000
